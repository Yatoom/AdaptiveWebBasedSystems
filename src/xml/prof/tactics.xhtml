There are quite some tactics that can be useful while playing chess. The table below mentions them all briefly. Read all sections to become a tactical wizard.

Name Description
A fork occurs when a piece attacks two or more pieces at the same time.
A pin occurs when a piece is attacked and if it moves a piece of greater value will be under attack.
A skewer is similar to a pin except in this case the piece of greater value (for example, the King, Queen or Rook) is in front.
A discovered attack occurs when a piece moves and in doing so it uncovers an attack on an enemy piece. If a discovered attack gives check to the enemy King it is known as a discovered check.
Removing the defender leaves the piece it was defending vunerable to capture.
Sometimes a chess piece is overloaded by having more than one defensive job to do.
Deflection occurs when a piece is deflected away from defending a piece or important square.
A decoy occurs when a piece is decoyed onto a fatal square.
